# Homework Planner

Arts & Media School Islington - Homework Planner Application

## Quick Start

### Local Development

1. Install dependencies:
```bash
npm install
```

2. Run the server:
```bash
node server.js
```

3. Open http://localhost:3000 in your browser

### Deploy to Render

See **RENDER_SETUP.md** for detailed instructions!

**Quick steps:**
1. Push to GitHub
2. Go to https://render.com
3. Sign up with GitHub
4. New → Web Service → Connect your repo
5. Build Command: `npm install`
6. Start Command: `node server.js`
7. Deploy!

## Features

- ✅ Student view of homework calendar
- ✅ Teacher login and dashboard
- ✅ Create, view, and delete homework
- ✅ Filter by year group and subject
- ✅ Download PDF of weekly homework
- ✅ Clickable, expandable homework descriptions

## Tech Stack

- Node.js
- Express
- Express-session
- Vanilla JavaScript (frontend)

## File Structure

- `server.js` - Express server
- `public/index.html` - Frontend application
- `homework.json` - Homework data storage
- `teacher_logins.txt` - Teacher credentials
